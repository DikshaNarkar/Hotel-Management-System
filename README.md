# 🏨 Hotel Management System  

## 📖 Project Overview  
The **Hotel Management System** is a **C++ project** designed to streamline hotel operations. This system enables hotel staff to efficiently manage **room bookings, guest check-ins, check-outs, customer records, and billing**. By automating these tasks, the system ensures improved service quality and operational efficiency.  

## **✨ Features**  
✅ **Room Management** – Add, search, and display available rooms.  
✅ **Guest Check-in** – Record guest details and update room status.  
✅ **Guest Check-out** – Calculate bill based on stay duration and advance payment.  
✅ **Customer Search** – Find a guest using their name.  
✅ **Billing System** – Auto-generate total payable amount for guests.  
✅ **Guest Summary Report** – View details of all checked-in guests.  
✅ **Menu-Driven Interface** – Easy-to-use system for staff.
